## Como rodar?

`git clone https://github.com/Ryannnkl/AirCnC.git`

### <code>Backend</code>

```bash
cd backend

npm install

npm run dev
```

### <code>Frontend</code>

```bash
cd frontend

npm install

yarn start
```

### <code>Mobile</code>

```bash
cd mobile

npm install

expo start
```
